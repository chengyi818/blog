如何阅读一本书

Posted on 2017-11-26   |   In 读书笔记   |  

阅读条件:

适合喜欢阅读的朋友

前言

最近读了一本经典书籍,如何阅读一本书.

相比于不读书而言,读书肯定是要好很多的.但是我们读书的目的是什么呢?书中分了三种情况:

  1. 为了娱乐.
  2. 为了资讯信息.
  3. 为了增进理解.

为了娱乐,就是读小说,笑话之类的.纯粹是一带而过.难度最小.为了资讯典型的就是指阅读新闻,现在流行地泡沫阅读也算是.难度次之.
最困难的是为了增进理解力,比如阅读复杂的技术书籍,这需要记忆练习.需要在你的脑海里建立新的神经通路,是有些痛苦的.而本书就是帮助读者阅读这些比较困难的书的.

说说我自己吧,我也算挺爱看书的吧.但是对于比较复杂的技术书籍,总感觉读在表面上,没有吸收.读了这本书,真的有点醍醐灌顶的感觉.之前读书真的是,把书读成了线性的了.其实读书,更应该是立体的.我们下面慢慢来说.

主动阅读

阅读不但不是一个被动的过程,反而需要大量的思考.不但要去理解作者的架构和内容,同时还需要在内心中构建自己的思考模型,最后还需要和现实中遇到的情况互相印证,来检验和调整模型.

阅读的效果在于主动的提出问题.书中提出了阅读每本书都要提出的4个基本问题:

  1. 架构性: 整本书谈的是什么?
  2. 诠释性: 关键词,关键句,关键逻辑过程是什么?
  3. 评论性: 作者讲的有没有道理?
  4. 和我的现实生活,已有知识结构有什么关系?

阅读的分类

书中将阅读的能力分为了4种:

  1. 基础阅读
  2. 检视阅读
  3. 分析阅读
  4. 主题阅读

基础阅读

指的是理解一句话字面含义的能力.对于母语,我们一般在小学初中就已经具备了这样的能努力.但是对于外语书籍相对来讲会吃力些.

检视阅读

检视阅读分为两种:

  1. 阅读前言,目录和每个章节的前后重点.指的是在尽可能短的时间内获取书籍的基本信息.重点是查看这本书是否值得阅读.
  2. 快速浏览整本书的内容,对书的架构做到心中有数.同时呢,对书的侧重难点也有所把握.遇到暂时不理解的内容,应该跳过去,不要纠缠.

这部分是我之前做的很差的.

首先,我经常是拿到书就读.没有对整本书的架构有所把握.其次,对于稍微困难的部分,也经常纠缠,既浪费时间,也容易打击信心.

分析阅读

这个就是一般意义上的细读,精读了.

这部分是书中的重点.需要不断提出问题,后面我们还会谈到.

主题阅读

最后这个主题阅读,是阅读的最高层次.

一般来讲,我们是通过阅读一本书来获得知识.整本书往往会谈到多个主题.但是如果是关于一个主题呢?比如我前几周看的字符编解码相关的知识.

往往需要阅读更多的资料,相互融会贯通.用自己的语言把知识描绘出来,才能算得上了解了一个主题.

分析阅读

首先,分析阅读建立在检视阅读的基础之上,我们会花费比较多的精力来进行分析阅读,所以书籍本身必须值得这样的花费.

再次强调,阅读的重点在于主动的问题.分析阅读主要细化了4个基本问题.

架构性问题

规则1: 书籍分类

不同类型的书,阅读方式和侧重点有所不同.我目前重点看的是IT技术书籍,是归为实用工具类书籍.

规则2: 主题

用简短的语言,概括整本书讲的内容.

规则3: 纲要

可能的话,尽可能详细的描述书籍的大纲.思考作者是用什么样的大纲来阐述自己的主题.

规则4: 提炼问题

作者想要在书中解决哪些问题?又对哪些问题有所保留?

诠释性问题

规则5:关键词 && 规则6:关键句 && 规则7:关键逻辑推理

整本书,每个章节都有自己的关键内容.需要自己去抽取出来.品味其中的逻辑.

规则8:问题解决的如何

这个是呼应规则4的.在规则4中,我们总结了作者将会涉及的问题.这里我们需要问自己,作者是否将这些问题解释清楚了.逻辑推理是否合理.

评论性问题

规则9:确保自己掌握了书中内容 && 规则10:不要带有预设立场,情绪化 && 规则11:反驳需要讲究逻辑

这三条放在一起,更多的是评论一本书之前的礼节性问题.

规则12: 作者的前提不够完整

规则13: 作者的前提不正确

规则14: 作者的推理不正确

规则15: 作者的论述不够完整

规则12~15,是比较正确的评论书籍的方式.在完全掌握内容的前提下,我们深入的评论作品有助于加深对内容的了解.将阅读到的替代经验内化.

关于自己

最后一个部分是关于学习到的内容和经验如何实践的部分.

如果书中讲了一个道理,你表示认同.那么正确的认同方式就是去实践它.这和知行合一的道理是一样的.

另外,学习技术方面的知识,我现在感觉还是要有一个主题.然后慢慢形成网络.避免泡沫阅读的陷阱,积极思考.

总结

我最近在阅读Python源码剖析,有的章节还是挺困难的.但是用书中的方法阅读,真的有蛮大的帮助.

在刚开始接触这样的阅读方法的时候,需要不断地刻意练习.直到可以成为本能.

不是所有的书籍都值得分析阅读.对于不是当前不是很重要的书籍,进行快速的检视阅读.在尽可能短的时间内,吸收尽可能多的知识,也不失为一件美事.
同时,我们对书籍的内容纲要有所掌握,在今后需要找寻相关资料时,也有裨益.

另外,我有一点小心得.上面提出的4个基本问题,其实不但对整本书适用,在阅读每一个章节的时候,也可以问自己这些基本问题.这不正是递归的思想么?
